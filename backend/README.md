# Backend

## ENV
In `.env` format:
```
ConnectionString__Database=
Logging__LogLevel__Default=
Logging__LogLevel__Microsoft.AspNetCore=
AllowedHosts=
```

In default `.json` format:
```
{
  "ConnectionString": {
    "Database": ""
  },
  "Logging": {
    "LogLevel": {
      "Default": "",
      "Microsoft.AspNetCore": ""
    }
  }
}
```
